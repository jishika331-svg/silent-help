# Silent Help
### An Anonymous Student Support Platform
Silent Help is a web-based anonymous platform designed to help students share personal, academic, or mental health concerns without fear of judgement. The platform encourages early support-seeking by providing a safe and private digital space.

---

## Problem Statement
Many student experience stress, harassment, academic pressure, or personal difficulties but hesitate to seek help due to fear of exposure, judgement, or lack of trust in existing systems. As a result, problems often remain unreported and unresolved.

---

## Solution
Silent Help provides a simple and anonymous web application where students can:
- Share their concerns without revealing identity.
- Select the category of their problem.
- View previously submitted anonymous posts.
- Access basic support and guidance resources.

This platform promotes mental well-being and emotional support in educational institutions.

---

## Features
- Anonymous problem submission.
- Category-based issue selection
  - Stress
  - Academic pressure
  - Harassment
  - Fianacial issues
- Real-time  display of submitted problems.
- Simple and user-freindly interface.
- Secure cloud-based data storage.

## How It Works
1. User opens the Silent Help web application.
2. User submits a problem anonymously.
3. The data is stored securely in Firebase Firestore.
4. Submitted problems are fetched and displayed in real time.
5. Users can view shared concerns and available support resources.
